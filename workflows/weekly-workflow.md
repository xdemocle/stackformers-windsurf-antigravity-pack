# Weekly Workflow

## Every Friday

1. Review completed tasks
2. Update memory.md with progress
3. Plan next week's priorities
4. Check Lighthouse scores
5. Update rules.md if new patterns emerged

## After Major Milestones

1. Update project structure in memory.md
2. Document new integrations
3. Add any new design patterns to rules.md
4. Create backup of key configurations

## Project Health Metrics

Track weekly:

- [ ] Lighthouse performance score
- [ ] Number of completed features
- [ ] Number of open bugs/issues
- [ ] Bundle size
- [ ] Page load times
